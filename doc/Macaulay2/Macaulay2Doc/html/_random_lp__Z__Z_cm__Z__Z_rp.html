Macaulay2Doc :: random(ZZ,ZZ)

random(ZZ,ZZ) -- random integer in a range

Synopsis

  • Function: random
  • Usage:
    random(min,max)
  • Inputs:
    • min, an integer
    • max, an integer
  • Optional inputs:
    • Density => ..., default value 1, make a random module map
    • Height => ..., default value 10, random element of a type
    • MaximalRank => ..., default value false, make a random module map
    • UpperTriangular => ..., default value false, make a random module map
  • Outputs:
    • an integer, a random integer in the range min .. max

Description

i1 : for i to 10 list random(100,200)

o1 = {124, 165, 171, 172, 119, 119, 191, 172, 193, 179, 172}

o1 : List
 
i2 : tally apply(100, i -> random(10,15))

o2 = Tally{10 => 13}
           11 => 20
           12 => 15
           13 => 20
           14 => 16
           15 => 16

o2 : Tally
