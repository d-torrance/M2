Macaulay2Doc :: degreeLength

degreeLength -- the number of degrees

Synopsis

  • Usage:
    degreeLength X
  • Inputs:
    • X, a ring, an ideal, a module, or a monoid
  • Outputs:
    • an integer, the length of a multidegree vector used as a degree of an element of X

Description

i1 : degreeLength ZZ

o1 = 0
 
i2 : degreeLength (ZZ[x])

o2 = 1
 
i3 : degreeLength (ZZ[x, Degrees => {{1,2,3}}])

o3 = 3
 
i4 : degreeLength (GF 9)

o4 = 0
