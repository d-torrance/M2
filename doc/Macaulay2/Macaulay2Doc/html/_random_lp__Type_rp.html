Macaulay2Doc :: random(Type)

random(Type) -- random element of a type

Synopsis

  • Function: random
  • Optional inputs:
    • Density => ..., default value 1, make a random module map
    • Height => ..., default value 10
    • MaximalRank => ..., default value false, make a random module map
    • UpperTriangular => ..., default value false, make a random module map

Description

Synopsis

  • Usage:
    random T
  • Inputs:
    • T, a type
    • Height => ...
  • Outputs:
    • a random instance of the type T. If the Height option specifies a number h and T is ZZ and , then the integers returned are in the range 0 .. h-1; for QQ the numerator and denominator are in the range 1 .. h.

Synopsis

  • Usage:
    random(d,R)
  • Inputs:
    • d, an integer or a list, the degree or multi-degree to use
    • R, a ring
  • Outputs:
    • a random homogeneous element of the ring R of degree d 
i5 : R = ZZ[x,y];
 
i6 : random(5,R)

       5     4      3 2     2 3       4     5
o6 = 3x  + 6x y + 9x y  + 3x y  + 4x*y  + 8y
